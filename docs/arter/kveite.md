# 🐟 Kveite (Hippoglossus hippoglossus)

> Premium matfisk - ein av dei verste kampfiskane i norske farvatn

## Kjenneteikn

| Kjenneteikn | Beskrivelse |
|-------------|-------------|
| **Størrelse** | 50-200cm (kan bli opptil 3m!) |
| **Vekt** | 5-50kg (kan bli opptil 300kg) |
| **Fargar** | Mørk brun/grønn rygg, kvit buk, kvite prikkar |
| **Kjenneteikn** | Flat, venstre auger på oversida |

## Kor finn ein kveite?

### Langs heile kysten

| Område | Beste tid | Korleis? |
|---------|-----------|----------|
| **Lofoten** | Mai-Sept | Båt (charter) |
| **Vestlandet** | April-Sept | Båt |
| **Sørlandet** | Mai-Sept | Båt, teine |
| **Oslofjorden** | Sommer | Båt, tungt |

## Når er kveite på?
