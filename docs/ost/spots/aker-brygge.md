## 🎣 Korleis fiske

### For byrjare

**Det du treng:**
1. Henglar-sett (199kr)
2. Markagg (49kr)
3. Tid og tålmod

**Steg:**
```
1. Gå til kaien ved Aker Brygge
2. Fest mark på kroken
3. Kast ut mot fjorden
4. Vent 10-30 minutt
5. Kjenn etter biting - slå til!
```
