## 🐟 Fisk i Oslofjorden

Oslofjorden er Norges mest tilgjengelige fiskefjord - berre 15 minutt frå Oslo sentrum!

| Art | Sesong | Vanskeleg | Kvar? |
|------|--------|-----------|-------|
| **Torsk** | Hele året | ★☆☆ | Kaiene |
| **Sei** | Sept-Mai | ★☆☆ | Dypt vatn |
| **Makrell** | Juni-Sept | ★★☆ | Fjorden |
| **Abbor** | April-Sept | ★☆☆ | Indre fjord |
| **Gjedde** | Mars-Juni | ★★☆ | Skarpa, grunt |

### Indre Oslofjord

**Grunna:**
- **Huk/Bogstad** - Klassisk abbor og gjeds
- **Søndre Åland** - Båt
- **Nordre Åland** - Båt

### Ytre Oslofjord

**For meir seriøse fiskere:**
- **Drøbak** - Torsk, sei
- **Hvitsten** - Torsk
- **Svelvik** - Torsk

## 🎣 Teknikker

### For byrjare

**Aker Brygge:**
```
1. Kjøp henglar-sett (199kr)
2. Fest mark på kroken
3. Kast ut frå brygga
4. Vent til du kjenner biting
5. Slå til og vik inn!
```

### For vidrekomne

**Gjedde i indre fjord:**
```
1. Bruk wobbler (stor)
2. Fiske grunt (1-3m)
3. Sakte innsveivjing
4. Følg med på tviklar!
```
