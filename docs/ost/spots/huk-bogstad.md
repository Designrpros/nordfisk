## 🎣 Korleis fiske

### For abbor (beste for byrjare!)

```
1. Fest mark eller wobbler på kroken
2. Kast inn mot siv/kantlys
3. Vik inn sakte
4. Følg med på tviklar!
5. Slå til når abboren biter!
```

### For gjedde

```
1. Bruk stor wobbler
2. Kast grunt (1-3m)
3. Vik inn i sakteste tempo
4. Gjedda tek ofte når du trur det er tuast!
```
