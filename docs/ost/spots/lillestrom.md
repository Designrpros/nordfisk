**Tog** er absolutt enklaste!

## 🎣 Korleis fiske

### For gjedde

```
1. Bruk stor wobbler (12-15cm)
2. Kast mot siv og vasskant
3. Vik inn veldig sakte
4. Følg med - gjedda tek ofte på "daudt" agn!
5. Vera klar for sterk kamp!
```

### For abbor

```
1. Fest mark eller liten wobbler
2. Kast mot bryggene
3. Vik inn i jamn fart
4. Abboren tek gjerne når du vik inn fort!
```
