# 🐟 Sei (Pollachius virens)

> Den nest vanligaste torskefisken - lett å få og god mat

## Kjenneteikn

| Kjenneteikn | Beskrivelse |
|-------------|-------------|
| **Størrelse** | 30-70cm (kan bli opptil 1m) |
| **Vekt** | 1-5kg |
| **Fargar** | Mørk grøn-blå rygg, sølvaktige sider, kvit buk |
| **Kjenneteikn** |rett munn, tydeleg sidelinja |

## Kor finn ein sei?

### Langs heile kysten

| Område | Beste tid | Korleis? |
|---------|-----------|----------|
| **Oslofjorden** | Sept-Mai | Båt eller kaifiske |
| **Bergen** | Hele året | Kaiene |
| **Stavanger** | Sept-Mai | Båt |
| **Nord-Noreg** | Hele året | Kaifiske |

## Teknikker

### 1. Pilking (best)

```
1. Fest pilot/sluk på kroken
2. Kast ut og la den sinke til botn
3. Vik inn sakte med bevegelser
4. Følg med på tviklar
5. Slå til!
```

### 2. Dorging

```
1. Fest agn på fleire kroker
2. Slepp ut 50-100m line
3. Kjør sakte medan båten går
4. Følg med på fleire liner
5. Trekk inn ved biting!
```
