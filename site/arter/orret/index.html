    🐟 Ørret (Salmo trutta)

    Den vanligaste ferskvannsfisken - finns i heile Noreg

    Kjenneteikn

    Kjenneteikn Beskrivelse
    Størrelse 20-50cm (kan bli opptil 10kg)
    Vekt 0.2-2kg (kan bli større)
    Fargar Brun/grå rygg, gull/brune sider, raud/brune prikkar
    Kjenneteikn Fete finner, karakteristiske prikkar

    Typar av ørret

    Type Kvar? Kva?
    Sjøørret Langs kysten Går til sjøs, kom tilbake
    Ferskvannsørret Innsjøer, elver Berre ferskvann
    Bekkeørret Små bekker Liten, men god mat
    Store ørret Djupe innsjøer Kan bli 10kg+

    Kva treng du?

    Budsjett (200-500 kr)

    Produkt Pris Kva det er
    Spinnstang 6ft 299kr For byrjare
    Fiskesnøre 49kr Mono 0.25mm
    Spinner 79kr For ørret

    Mellomklasse (500-1500 kr)

    Produkt Pris Kva det er
    Spinnstang 7ft 699kr Meir kontroll
    Fluestang 999kr For fluefiskere
    Vadarar 799kr For elvefiske

    Teknikker

    1. Spinnerfiske (best for byrjare)

    1. Fest spinner på lina
2. Kast på tvers av elva
3. Vik inn med jamn fart
4. Følg med på tviklar
5. Slå til!

    2. Fluefiske

    1. Velg riktig flue
2. Kast på tvers
3. Lat flua sanke nedstraums
4. Følg med på snøret
5. Slå til når ørreten tar!

    3. Wobbler

    1. Fest wobbler (liten)
2. Kast inn mot land
3. Vik inn sakte
4. Følg med på tviklar
5. Slå til!

    ⚖️ Reglar

    Regel Verdi
    Minstemål 30cm (sjøørret), 20cm (ferskvann)
    Dagleg kvote Varierer
    Fredningstid Varierer etter vassdrag
