# 🐟 Torsk (Gadus morhua)

> Norges nasjonalfisk! Lett å få, god å eta, og du finn den overalt langs kysten.

## Kjenneteikn

| Kjenneteikn | Beskrivelse |
|-------------|-------------|
| **Størrelse** | 40-100cm (kan bli opptil 1.5m) |
| **Vekt** | 1-10kg (kan bli opptil 50kg) |
| **Fargar** | Grå-grønn rygg, kvit buk, mørke flekker |
| **Hovud** | Stort med tydeleg skjeggbar |
| **Finner** | Tre ryggfinner, to gatfinnar |

## Kor finn ein torsk?

### Geografisk

Torsk finn ein langs **heile norskekysten**, frå Oslofjorden til Nordkapp.

| Område | Beste tid | Vanskeleg |
|--------|-----------|-----------|
| **Sørlandet** | Mars-Mai | ★☆☆ |

| **Trøndelag** | April-Juni | ★☆☆ |
| **Lofoten** | Jan-Mai | ★★☆ |
| **Troms** | Nov-Mars | ★☆☆ |

### Typer av steder

- **Kaier** - Alle kommunale kaier er gylne! Prøv Bryggen i Bergen, Aker Brygge i Oslo
- **Strender** - Sandstrand fungerer bra
- **Båt** - For meir seriøst fiske, leig båt eller charter
- **Fyr** - Mange fyr er tilgjengelege for overnatting + fiske

## Teknikker

### 1. Botnfiske med mark (best for nybyrjare)

```
1. Fest mark på kroken
2. Kast ut til det kjennest på botn
3. Vik inn line til du kjennest "biting"
4. Slå til og start å vikja inn!
```

### 2. Pilking (meir aktiv)

```
1. Fest pilk/kunstig agn
2. Løft stanga opp-ned i rytme
3. Følg med på tviklaren - når den snor = biting!
4. Slå til!
```

### 3. Dorging (fra båt)

```
1. Fest agn på fleire kroker (maks 6 per person)
2. Slepp ut line medan båten går sakte
3. Følg med på teiner/tenner
4. Trekk inn når du kjennest biting
```

## Oppskrift: Steikt torsk

### Ingrediensar

| Mengd | Ingrediens |
|--------|------------|
| 400g | Torskfilet |
| 2 ss | Smør |
| 1 ts | Salt |
| 0.5 ts | Pepper |
| 1 stk | Sitron |

### Metode

1. **Forvarm** steikepanna på medium varme
2. **Smelt** smøret
3. **Krydre** fileten med salt og pepper
4. **Steik** 3-4 min på kvar side til kjøtet er kvit og flaknar lett
5. **Server** med kokte poteter, smelta smør og sitron

### Tips

- 🔥 Ikkje for varmt - torsk er tørt viss det steikest for lenge
- 🧈 Meir smør = meir smak
- 🍋 Sitronmask kompenserer for tørke

## 🐟 Forviklingar

- **Langelandstorsk** - same art, namnet kjem frå Danmark
- **Skrei** - torsk som gyter i Lofoten (sesong: Jan-Mars)

## 🎣 Gangsteg for ditt første torskefiske

1. Kjøp eit billig henglar-sett
2. Grav mark (eller kjøp på bensinstasjon)
3. Gå til nærmaste kai
4. Fest mark på kroken
5. Kast ut, vent, og nyt!
6. Når du kjennest biting - slå til!
7. Vikk inn og beundre din første torsk!
