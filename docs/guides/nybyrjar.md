# 🎣 Nybyrjarguide

> Aldri fiska før? Ingen problem! Denne guiden tar deg frå null til din første fangst.

## 🐟 Kva er fiske eigentleg?

Fiske går ut på å lure ein fisk til å bita på ein krok med mat. So enkelt er det.

### Kva treng du?

1. **Fiskestong** - for byrjere anbefalar me ein ferdig "henglar-sett"
2. **Krok** - fest på enden av linja
3. **Agg** - mat for å lure fisken (mark, reke, eller kunstig)
4. **Fjærlett** - for å halde kroken i djupna
5. **Søkke** - for å vege ned linja

---

## 🎯 Dei tre viktigaste tinga

### 1. Kva tid?

| Tid på døgnet | Kva fisk? |
|----------------|
| **Morgon (06-09)** | Beste tida! Fisk er aktiv då. |
| **Kveld (17-21)** | Ogå godt. |
| **Natt** | Laks og sjøørret - butikk! |

### 2. Kvar?

- **Kaier** - Alle kaier langs kysten er gratis å fiska
- **Strender** - Godt for sjøfiske
- **Innsjøer** - Fredagstur-favoritt
- **Elver** - Følg laksefiskere for tips

### 3. Kva fisk?

Start med **torsk** eller **sei** - dei er:
- Lett å få
- Finnes overalt
- God mat

## 🐟 Kva gjer du når du får ein fisk?

### Steg 1: Sjekk minstemål

| Art | Minstemål |
|-----|-----------|
| Torsk | 32cm |
| Sei | 30cm |
| Laks | 35cm |
| Ørret | 30cm |

**For liten?** Sett den forsiktig tilbake i sjøen! 🐟

### Steg 2: Slå til!

Drep fisken hurtig ved å:
1. Slå hovudet mot noko hardt
2. Skjær opp magen
3. Fjern innmaten
4. Skyll i saltvatn
5. Oppbevar kaldt

### Steg 3: Steik!

Einklas oppskrift:

```
1. Skjær i fileter
2. Mel på båe sider
3. Steik i smør på medium varme
4. 3-4 min kvar side
5. Server med sitron og potet
```

---

## ⚠️ Viktige reglar

1. **Fiskekort** - Mange stader krev fiskekort (50-150 kr/dag)
2. **Minstemål** - Sjekk før du tek med
3. **Fredningstid** - Lakselvar har fredningstid om våren
4. **Lisens** - For laksefiske treng du Fiskerlisens
