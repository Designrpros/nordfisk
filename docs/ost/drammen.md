## 🚤 Korleis fiske i Drammen

### Elvefiske (sjøørret/laks)

**Treng:**
- Fluestang eller spinnestang
- Fiskekort (300kr/dag)
- Fiskerlisens

### Kaifiske (torsk/sei)

**Gratis!**
```
1. Gå til Drammen havn
2. Fest mark på kroken
3. Kast ut frå kaien
4. Vent 10-20 minutt
5. Nyt!
```
