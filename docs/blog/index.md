## ✍️ Skriv din eigen rapport

Har du vore på ein artig fisketur? Send inn din rapport!

### Korleis skriva ein fangstrapport

1. **Kvar** - Stad og tidspunkt
2. **Kva** - Fisk du fekk (art, storleik, vekt)
3. **Korleis** - Teknikk, agn, utstyr
4. **Tips** - Kva ville du gjort annleis?
