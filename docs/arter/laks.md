# 🐟 Laks (Salmo salar)

> Kongen av norske fisk - ein eventyrfisk

## Kjenneteikn

| Kjenneteikn | Beskrivelse |
|-------------|-------------|
| **Størrelse** | 50-100cm (kan bli opptil 150cm) |
| **Vekt** | 3-10kg (kan bli opptil 30kg+) |
| **Fargar** | Sølv-blå rygg, kvite flekker under sidelinja |
| **Kjenneteikn** | Svarte prikkar over sidelinja, raud-bru prikkar |

## Teknikker

### 1. Fluefiske (klassisk)

```
1. Velg riktig flue for sesongen
2. Kast på tvers av elva
3. Lat flua drive nedstraums
4. Følg med på snøret
5. Slå til ved biting!
```

### 2. Spinnerfiske (for byrjare)

```
1. Fest spinner på lina
2. Kast på tvers
3. Vik inn med jamn fart
4. Følg med på tviklar/genre
5. Slå til!
```
